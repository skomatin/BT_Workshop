#### Useful Keyboard shortcuts:

Arduino IDE :

Uploade code to the Board: CMD + U
Compile code             : CMD + R
Open Serial Monitor      : CMD + SHIFT + M

XCODE: 

Run simulation           : CMD + R
Build code               : CMD + B
